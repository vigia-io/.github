# Contributing

## Issues

You are welcome to [open issues](https://github.com/vigia-io/.github/issues) for questions, documentation feedback, and general product discussion.

## Pull requests

**Pull requests are accepted only from repository collaborators** (Vigia team members).

Forking is disabled on our public repositories, so external contributors cannot submit PRs via fork. If you have a commercial agreement that includes contribution rights, contact **legal@vigia.io**.

## Internal development

Product code lives in private organization repositories. Releases follow the team's internal CI/CD process.
